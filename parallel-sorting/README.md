# Параллельные сортировки  
Задание 1 по дисциплине **«Параллельные и распределённые вычисления»**

## Описание
Приложение демонстрирует параллельное выполнение трёх алгоритмов сортировки:
- Пузырьковая сортировка
- Быстрая сортировка
- Сортировка слиянием

Для каждого алгоритма выводятся:
- Время выполнения
- Количество сравнений
- Количество обменов (перемещений)

Массив генерируется случайно, все три алгоритма работают с **одним и тем же набором данных**.

## Технологии
- C#
- Windows Forms
- `System.Threading.Thread`

## Запуск
1. Открыть файл `ParallelSortingApp.sln` в Visual Studio 2017+
2. Нажать **F5** или **«Запуск»**

> Проект не требует сторонних зависимостей.
